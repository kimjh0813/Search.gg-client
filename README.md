### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.



# 디렉토리 구조

- src
  - api
    > api call 함수
  - assets
    > 이미지, 아이콘 등 정적 파일
  - components
    > 컴포넌트
  - hooks
    > custom hooks(react-query 등)
  - pages
    > 화면을 구성하는 page
  - reduce
    > redux 코드
  - types
    > response model, util type 등의 type
  - utils
    > global function
